# Sistema de Frota

MVP web responsivo para gestão de veículos, motoristas, viagens, abastecimentos e manutenções.

## Como executar

Abra o arquivo `index.html` no navegador. Os dados são salvos localmente no navegador com `localStorage`.

## Funcionalidades

- Dashboard com indicadores e alertas
- Cadastro e busca de veículos
- Cadastro de motoristas e controle de validade da CNH
- Registro de viagens
- Controle de abastecimentos e custos
- Manutenções preventivas e corretivas
- Layout responsivo para celular e computador

## Publicação

O projeto é estático e pode ser publicado diretamente no GitHub Pages.
